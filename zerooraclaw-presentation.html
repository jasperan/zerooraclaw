Why Oracle?

In-DB ONNX Embeddings

Generate vectors inside the database. No external embedding service. Lower latency, zero network hops.

AI Vector Search

Native vector indexing with HNSW and IVF. Hybrid search combining FTS + vectors with RRF scoring.

ACID Guarantees

Full transactional isolation for multi-agent workloads. No eventual consistency surprises.

Multi-Agent Isolation

Session-level partitioning. Each agent gets its own namespace, history, and memory scope.

26ai Free Tier

Always-free Oracle Cloud tier. Run production workloads at zero cost with ARM A1.Flex instances.

Pure Rust Driver

No InstantClient dependency. Compiles and runs anywhere Rust does, including Alpine containers.

Safety & Security

Defense in depth, built into every layer

15

Defense in Depth

Prompt Injection Detection

Multi-layer scanning catches injection attempts before they reach the LLM. Extracted to its own crate.

Trust-Based Attenuation

Skills and tools get different capability ceilings. Installed extensions can't access destructive tools.

Approval Gates

Destructive operations require explicit user approval. Configurable per-tool and per-policy.

Input Validation

Schema-driven validation on every tool argument. Rejects malformed input before execution.

Data Leak Detection

Scans outbound responses for sensitive data patterns (PII, secrets, credentials).

Sandbox Isolation

Docker-based execution sandbox with network proxy, domain allowlists, and credential injection.
